## PageTurner
This is a tool written in Go with FFMPEG CLI that I am using to convert audio-books from mp3 to m4b.

### Build requirements

- Go 1.16

### Run requirements

- ffmpeg
- ffprobe

### Building

    make

Or

    go build

### Installing on Linux and Mac

1. Make sure you have prerequisites installed
2. Build and execute

        make install
